Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the dom. They are listed below :

  1. 1. document.getElementById() - Selects an element by its ID.
  2. 2. document.getElementsByClassName() - Selects elements with a class name.
  3. 3. document.getElementsByTagName() - Selects elements by a tag name.
  4. 4. document.querySelector() - Selects only the first matching element that contains a given css selector
  5. 5. document.querySelectorAll() - Selects all matching elements that contains a given css selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

  • 1.innerHTML - It is used for getting an element's inner HTML structure and also for changing or setting an elements innerHTML structure.
  • 2.innerText - It is used for getting an element's inner text contents which are not hidden and also for changing or setting an element's inner text.
  • 3.textContent - It is used for getting an element's inner text contents (doesn't matter hidden or not) and also for changing or setting an element's inner texts.

Question-3: What is event delegation in the DOM?

Event delegation refers to the method where an event listener is attached to a parent element instead of attaching it directly to the child element, which is the source of the event. Event delegation takes advantage of event bubbling, where events moves upwards from the child element to the parent to find the source of the occurred event.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and then bubbles up to its parent elements. It allows parent elements to manage events for multiple child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create an element - document.createElement("tagName") used for creating a new element.
  • Add an element - parentElement.appendChild(newElement) to add it to a parent element.
  • Remove an element - parentElement.removeChild(element) is used to remove a child element from a parent element.
